.. image:: resource/widget/PGForm.png

폼 Form
----------

.. thumbnail:: resource_new/form.png

폼 위젯입니다.
Input 요소를 감싸주는 <form> 태그 위젯입니다. 따라서, Input 요소를 추가하기 전에 필수로 생성해야 합니다.
폼 위젯을 먼저 만들고 그 안에 Input 위젯을 넣어주세요.


**속성**

* 하위요소 : 가능
* 링크 : 불가능
* 스크롤 애니메이터 : 가능
* 배경이미지 : 가능
* 백엔드 엘립시스 : 불가능
